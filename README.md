# EasierMinicom

## What is EasierMinicom
EasierMinicom is a shell script to help you use minicom easier.

* auto detect devices, then list them for user to choose
* auto save log to /tmp

## How to use
```bash
echo "source $(pwd)/EasierMinicom.sh" >> ~/.bashrc
source ~/.bashrc
com
```
